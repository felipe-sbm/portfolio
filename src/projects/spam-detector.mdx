## Contexto

Esse projeto aplica **Machine Learning** para classificar mensagens e reduzir ruído na caixa de entrada.

## O que foi construído

- Pipeline de pré-processamento de texto
- Vetorização de mensagens
- Treinamento e avaliação de modelo de classificação
- Camada de inferência para classificar novas mensagens

## Stack

- C# .NET
- Bibliotecas de ML
- Dataset rotulado para treinamento

## Aprendizados

Aprendi bastante sobre qualidade de dados, balanceamento de classes e impacto de feature engineering no desempenho final do classificador.
